# 3D Data

## The Representation Challenge of 3D Deep Learning

- 오디오나 이미지와 같은 **rasterized form(regular grids)** 을 가진 데이터와 달리, 3D 데이터는 **geometric form (irregular)** 의 형태를 지니고 있음.
- 3D Data는 아래와 같은 형태로 나타낼 수 있음.
  - Point Cloud
  - Mesh (Graph CNN)
  - Volumetric (voxelize)
  - Multi-View
